# SyntaxHighlighter

SyntaxHighlighter es una aplicación web donde podrás administrar tus cursos como profesor y subir código referente al mismo, así como calificar con un sistema de rating a los comentarios de tus alumnos. Y a los alumnos poder visualizar los códigos compartidos por sus profesores y comentar acerca de ellos.

Utiliza la herramienta SyntaxHighlighter v3.0.83 de Alex Gorbatchev http://alexgorbatchev.com/SyntaxHighlighter


## Requisitos

- Java EE 8.0
- GlassFish Server 4.1.1
